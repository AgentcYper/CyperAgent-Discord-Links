## 📌 Features Highlight

| Feature | Description |
| :--- | :--- |
| **🎯 Random Invite Generator** | Generates random Discord invite codes with customizable character length |
| **🔄 Proxy Rotator** | Built-in proxy rotation system with 30+ proxies to avoid rate limiting |
| **✅ Availability Checker** | Checks if invite codes are available (404) or taken (200) |
| **📨 Webhook Embeds** | Sends beautiful Discord embeds with images when available codes are found |
| **🌈 Fancy Banner** | Colorful animated banner with watermark effect |
| **🛡️ Error Handling** | Auto-skip proxy errors and continue scanning without stopping |
| **📊 Real-time Counter** | Live counter showing total taken/used codes found |
| **⚡ Fast Multi-threading Ready** | Code structure ready for multi-threading implementation |
